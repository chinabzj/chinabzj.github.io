什么是Markdown？
Markdown是一种轻量的标记语言，通过符号以及特殊语法就可以对文本进行编辑，也就是说手不离开键盘就可以完成对文本的编辑，同时可以实时预览，日常写博客、简书用到的比较多，那么想使用这种语言就要了解它的基本语法以及一些高级用法，一起来看看吧。

一、标题

标题的设置使用“#”来完成

以此类推，很显然，通过#的个数来控制标题等级，非常方便。#之后最好加一个空格，更标准一点。

二、字体

（1）普通字体

不加任何符号，直接编辑文本，就是比较普通的字体。

（2）加粗字体

在想要加粗的文本两边用“**”包含起来，就可以加粗字体

效果就是这样：加粗

（3）斜体字体

在想要斜体的文本两边用“*”包含起来，就可以设置斜体

效果就是这样的：斜体

（4）斜体+加粗

显然以此类推，只需要用“***”将文本包含起来即可

效果就是这样：**斜体+加粗**

（5）删除

实现删除线功能，只需要用“~~”将文本包含起来

效果就是这样：我被删除了

（6）下划线

Markdown的语法与HTML兼容，所以可以HTML的标签实现，但是下划线可能让人误以为是超链接，所以尽量不要使用

效果就是这样：我是下划线

四、文本字体颜色字号

字体颜色,文本左边“<font color=””>”，右边“</font >”即可,字体就是“face”，字号“size”，和color一样设置，也可以加在前后，一起设置,如下

<font size=4 color=red face=”微软雅黑”>文本</font>

效果如下

我是红色三号字体微软雅黑

字号数字越大字体越大，颜色可以用英文（方便），也可以用十六进制颜色代码（要双引号），字体就直接打字即可（双引号）

（1）居中

（2）左对齐

（3）右对齐

七、插入图片（hexo博客为例）

插入图片有很多种方法，这里主要叙述两种方法

（1）本地图片

首先我们设置根目录的配置文件_config.yml,将post_asset-folder: false 改为post_asset-folder: true(注意冒号后面有空格)

然后安装插件：cnpm install https://github.com/CodeFalling/hexo-asset-image –save

然后我们执行 hexo n时在_posts文件夹下会生成同名文件夹，将要使用的图片放在这个文件夹里

添加图片只要写入

![描述性文字](图片名称.jpg)

即可，在hexo g执行后，会发现很多这样类似描述，就是写入你的图片，部署之后上传可能比较慢，请耐心等待

（2）网络图片

写如下代码即可

!描述文本

应该还有很多其他方法

八、转义字符

在Markdown中提供了转义字符，就比如说我想打<br>,我并不是想输出换行，仅仅只是打这个字符，怎么打出来呢

很简单，Markdown提供了转义字符“\”,只要在<>的左边加上这个转义字符即可，使用方法如下：

\<br\>
